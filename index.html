Trợ Giúp
Cách Sử Dụng:
  1. Nhấn nút "Chọn Thư Mục" để chọn thư mục chứa file PDF
  2. Hệ thống sẽ quét tất cả thư mục con và hiển thị cấu trúc cây
  3. Click vào file PDF để xem nội dung
  4. Sử dụng các công cụ zoom, chuyển trang để điều chỉnh xem
  5. Click vào biểu tượng sao để đánh dấu yêu thích
  6. Sử dụng search để tìm file nhanh
Lưu Ý:
  • Ứng dụng chạy 100% trên trình duyệt, không cần server
  • Dữ liệu được lưu trên máy của bạn (LocalStorage)
  • Chế độ Dark Mode được ghi nhớ
  • Danh sách yêu thích được lưu tự động
